# Invertibility
If $A$ is an $n \times n$ matrix. These conditions are equivalent.
1. $A$ is invertible.
2. The columns of $A$ are a basis for $\mathbb{R}^n$
3. Col $A = \mathbb{R}^n$.
4. Rank $A = \text{dim}(\text{Col } A)$ n.
5. Null $A = \{ \vec{0} \}$
